4. Calculation Flow

Step 1: Base Incentive = Revenue Achieved × Slab Rate
Step 2: Input Bonus = Base Incentive × Input Bonus %
Step 3: Penalty A = Base Incentive × Missed-Connect Penalty %
Step 4: Penalty B = Shortfall Multiplier × Shortfall Amount × Slab Rate (only if below 100%)
Step 5: Subtotal = Base + Input Bonus − Penalty A − Penalty B
Step 6: Penalty C = Subtotal × Closer Penalty %
Final: Payout = Max(0, Subtotal − Penalty C)
